### Task 4: 回填 AbilityManager MCP 懒加载 + 修改执行分支

- [ ] **Step 1: 启用 ListToolInfo 中的 MCP 懒加载代码**

将第 347-351 行被注释的代码启用，对齐 Python `list_tool_info` 第 506-518 行：

```go
// 4. MCP 懒加载：遍历 mcpServers，通过 ResourceMgr 获取 MCP 工具信息，
//    重命名为 mcp_{serverName}_{toolName}，注册到 am.tools 并追加到结果
for mcpServerName, mcpServer := range am.mcpServers {
    mcpServerID := mcpServer.ServerID
    mcpToolInfos, mcpErr := am.resourceMgr.GetMcpToolInfos(ctx, "", mcpServerID)
    if mcpErr != nil {
        logger.Warn(abilityLogComponent).
            Str("event_type", "mcp_lazy_load_error").
            Str("server_name", mcpServerName).
            Str("server_id", mcpServerID).
            Err(mcpErr).
            Msg("获取 MCP 工具信息失败")
        continue
    }
    for _, mcpToolInfo := range mcpToolInfos {
        mcpToolName := "mcp_" + mcpServerName + "_" + mcpToolInfo.GetName()
        mcpToolID := mcpServerID + "." + mcpServerName + "." + mcpToolInfo.GetName()
        // 创建 ToolCard 并注册到 am.tools
        mcpCard := tool.NewToolCardWithID(mcpToolID, mcpToolName, mcpToolInfo.GetDescription(), mcpToolInfo.GetParameters(), nil)
        am.tools[mcpToolName] = mcpCard
        // 重命名 ToolInfo
        mcpToolInfo.SetName(mcpToolName)
        toolInfos = append(toolInfos, mcpToolInfo)
    }
}
```

注意：需确认 `schema.ToolInfoInterface` 是否有 `GetName()`/`SetName()`/`GetDescription()`/`GetParameters()` 方法，如果没有需适配。
